Projects

Graphs
I developed a class in C++ to construct a graph from a specifically formatted CSV file. I implemented and designed graph algorithms, such as BFS(), DFS(), Dijksta's() for finding the shortest path, and connected components to perform various tasks.

Huffman Encoding
I implemented a tool to compress and uncompress files using Huffman Coding. Inputting an arbitrary file, I used huffman compression to create a compressed version of the file and the same algorithm reversed to uncompress the file.
